Most days that's at Aquatronic — fountains, water features, DMX lighting. I write the control software, wire the panels, and end up on the customer call too.

### Things I've built
- A native Windows app in C++ (Win32) with a full ANSI E1.20 RDM stack — addresses and debugs DMX/RDM gear in the field.
- A Python engine that turns a component netlist into finished, rule-checked DWG panel schematics. No more redrawing the same wiring by hand.
- A stock-tracking web app in daily use on the floor — single file, no backend to babysit.
- OCR + a CRM pipeline for the deeply unglamorous job of getting handwritten trade-fair cards into a database.
